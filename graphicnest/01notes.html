Lecture 1

Vector VS Raster

The first thing you need to know is the difference between vector and raster graphic.

Vector graphics are built starting from simple shapes and, more in general, they can be represented by mathematical formulas. This means that every content you create this way can be enlarged at any size and it will not lose resolution. Therefore, we use this kind of graphics if we want to print something or if we are not sure of the final size of the object where they will be applied. Think about a logo: you could use it on a business card (very small) as well as on a billboard (very large).

On the other hand, raster graphics are based upon pixels. A raster image (e.g. a photo) contains much more information than a vector one, both for details and color depth. This kind of image is mainly used in photography, mock-ups and web/digital content.

A raster file is much bigger than a vector one.

 

Adobe Illustrator: an introduction

Adobe Illustrator is the most famous software for the creation of vector graphics. You won’t be able to find a more complete software anywhere in the world. Unluckily, it’s quite expensive.

Free alternatives InkScape is quite good if you don’t need advanced functions.

Workspace

Let’s start by looking at the main interface.

First of all, let’s select “Essentials” from the drop-down menu in the up-right corner. This will setup all the main tools in the UI.

The menu on the left side of the screen is called toolbar, while the one on the right is the panels' menu.

Guess what, the toolbar contains all the main tools! The panels provide us with some very useful functions that we will discover later on.

The menu at the top (main menu) contains all the other advanced functions. Right below this menu we find the options’ bar. This allows us to set path’s properties and some tools’ options.

Creating a new document

In order to create a new document, we click File > New... (Ctrl+N). In the following dialog box we can insert the file’s name (otherwise we can do this while saving), we can decide the number of artboards (not mandatory) and set the size of the artboards. Click OK when you are ready.

Saving a document

There are two main ways to save:

  • Save (Ctrl+S) is the faster and will mostly save a .AI file.
  • Save as... (Ctrl+Shift+S) allows you to choose the format (e.g. .PDF)

Esportazione di un documento

There are different ways to export a document, depending on ho we plan to use it.

If you want it to be vectorial, so that you can print it, you must use one of the formats in the Save as... options (PDF is “universal”, SVG can be used in websites).

On the contrary, if you need a raster image of what you created, first you have to make sure that the subject is inside an artboard (we may have more than one in the workspace, the tool may be invoked by Shift+O). Then you click File > Export... and in the following dialog box you can select the most appropriate format (usually JPG or PNG) and the artboard(s) you wish to save (check Use artboards).

Navigating through artboards and workspace

Illustrator allows you to use a wide workspace, so that you can make drafts, and then you can organize what you did using artboards (that you will need also for the export).

There are many ways of navigating such a wide space:

  • Space (keep pressed) + Drag&Drop: to move around
  • Alt + MouseWheel: zoom in/out. Windows users: sometimes clicking alt enables the navigation through the menus, to avoid this you can press esc when you finish zooming.
  • Ctrl + + and Ctrl + -: zoom in/out
  • Zoom tool: you can select (by drag&drop) the area you want to zoom

Main tools

  • Selection tool (V) AKA Black Arrow: select, move and scale shapes and paths;
  • Direct Selection Tool (A) AKA White Arrow: select and move the anchor points of a path;
  • Pen (P) AKA Your best friend... maybe: fundamental tool of Illustrator (we will see it later on);
  • Text (T);
  • Line e Rectangle (M): right-clicking the rectangle tools shows the other available shapes;
  • Rotate(R) e Scale (S);
  • Shape Builder (Shift+M) AKA That's Magic!: allows you to perform boolean operations between shapes and paths;
  • Eyedropper (I): copies the style of a path (fill and stroke);
  • Hand (Space + drag&drop): workspace navigation
  • Zoom

Some icons have a triangle in the bottom-right angle: this means that by right-clicking it, you can unveil other tools.

In the panels’ menu you can find some important functionalities (alignment, pathfinder, layers, etc.)

Using the pen tool

Before speaking about the tool, let me explain better what a vectorial image is. It is composed by paths, that can be straight, curved, open or closed and are built on anchor points and handles, that allow us to modify the path’s shape.

The pen tool allows you to “draw” the anchors and handles in order to create paths.

To create a segment, you just need to click two different points (start and end).

If you want to draw a curve, all you have to do is clicking on the starting point, then you click and drag on the end point. By dragging, you will modify the point’s handles: the more you drag, the more the curve will be curved (sorry for the pun).

If you need to interrupt a curve, just click again on the last point.

Arts and Maths Handles are always tangent to the curve.

In general, paths are built with Bezier Curves*.

Boolean operations

The pathfinder panel and the shape builder tool allow you to perform boolean operations (union, subtraction, intersection) so that you can create complex shapes starting from basic ones.

In particular, the shape builder allows you to perform all the operations provided by the pathfinder, but in a more “powerful” way.

Using this tool is easy: you select all the shapes you need with the black arrow, then you press Shift + M to select the tool. You put the cursor on the first shape and you drag and drop on the areas that you want to unite/subtract/whatever (they will become highlighted). You repeat this until you reach your goal. If you need to select areas in a wide zone, you can also hold shift while dragging and dropping.

By default the tool works in combine/unite mode, so it will create new shapes and paths by union or intersection. If you want to subtract, you need to hold Alt while using it.

Working with texts

In order to type a text, you have to select the Text (T) tool. Now you have 4 different ways to proceed:

  • Click on an empty area in the workspace: this allows you to write freely on a line (you can wrap up your text with Enter)
  • Drag&drop: it allows you to create a text field, so the line wrap-up will be automatic
  • You can write on an oper path: put the cursor near it until you see a little wave symbol appearing near the T, then click and type.
  • If you want to have a text field with a “non-standard” shape, you can build one with the shape tools or the pen, then you take the text tool and you place the cursor inside the shape, near the border. When the cursor appearance changes, you can click and proceed as above.

After typing something, you can use the panels Character and Paragraph to change the style, kerning, appearance, alignment and size of what you typed.

After leaving the typing mode (esc), you can select the text with the black arrow, then you click Object > Envelope distort and select the first option (I don’t know how it is called in English, sorry). This should open a dialog box in wich you can choose the shape of the text (e.g. an arc or a wave).
