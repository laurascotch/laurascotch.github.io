Photo editing in Photoshop: basics

Using Photoshop

Photoshop's graphical interface can be divided into 4 main areas:

  • Menu bar: Various options and functionalities
  • Toolbar: on the left side of the monitor, there you can find all main tools. If an icon has a small triangle in the bottom right corner, you can right click it to discover more tools.
  • Options bar: It provides different options, based on the selected tool
  • Panels: On the right hand side of the monitor, there you can find various functionalities, one above all the Adjustments panel.

Non-disruptive editing

Editing an image in a non-disruptive way means that Photoshop allows you to perform operations that affect the appearance of the photo, without touching the original one. This is possible thanks to the use of layers.

One way of editing a photo involves the use of adjustment layers. Each one of these layers comes with a window in which you can customize them using sliders.

Adjustment layers are put above the image layer, so they do not alter it directly.

Sometimes it could be necessary that you directly edit the photo. In this case I suggest you duplicate the layer by selecting it and pressing Ctrl+J.

Open/Create a (new) document

As in any other software, you just need to click File > Open... or File > New.... If you want to edit a lot of images in batch, I suggest you use Adobe Lightroom (Photoshop's little brother).

Crop and straighten

Generally speaking, the first thing you want to do to improve your photos is straighten and resize/crop them.

Let's select the Crop (C) tool and make sure that Delete cropped pixels in the Option Bar isn't checked (so that you can achieve non-disruptive editing).

You can drag and drop the handles on the sides of the photo and crop it by pressing ENTER.

It is also possible to set up the crop proportions a priori from the option bar.

To straighten up a photo, with the crop tool selected, you need to click the Straighten icon on the option bar. Now you can draw, by drag and dropping, a line above the horizon of the photo. The photo will be straightened when you release the mouse click and it will mantain its original proportions. As usual, you confirm it by pressing ENTER.

Adjustment layers

Exposure

If a photo is too light or too dark, you need to fix it. There are 2 fast ways to do so:

  • Exposure equally adjusts all levels (lights, shadows, midtones) and it is a good tool if the photo il overexposed or underexposed "in its entirety"
  • Brightness/Contrast works more on midtones, without affecting lights and shadows too much. You can also modify the image's contrast, but there are better ways to do so

Color adjust

The main way to make your photos look better is to fix their colors.

Let's select the Vibrance panel. There you can modify both the Vibrance and the Saturation.

Saturation affects all colors. If you want a black and whit image, you can just set it to its minimum value (-100), otherwise it's better if you don't modify it.

Vibrance is smarter: it works just on the less saturated color, without modifying those that already stand out. It is not a good idea to go above +30, since the photo will look photoshopped.

Levels

Contrast

The best way (for newbies) to modify a photo's contrast is to use the Levels panel.

Edits done this way are based upon the istogram fo the photo, which describes the distribution of lights, midtones and shadows in the photo.

The easiest way to modify the contrast is to move towards the center of the slide the cursors for lights (the white one) and shadows (the black one). If this isn't enough, you can play also with the one for the midtones.

White balance

Again in the Levels window, you can see three eyedroppers: one for lights, one for midtones and one for shadows. Let us select the midtones' one and use it to click on a neutral color (best option is grey) in the photo. Photoshop will automatically perform the white balance.

Same thing holds for the other two eyedroppers. You just need to click on a particularly light (or dark) spot on the photo.

Undesired elements removal

It may happen that you wish to remove some elements from a photo: high tension cables, people, etc.

As a caution measure, we duplicate the image layer (Ctrl+J).

  • Deleting spots: you can use the Spot Healing Brush Tool (J). In the option bar you can set the size of the brush. To make it work, you need to cover an area that's slightly bigger than the spot.
  • Red eye correction: select the Red eyes tool and click on the red eyes (quite obvious, isn't it?).
  • Deleting big objects: after selecting the Lazo (L) tool or the rectangular selection (M) tool, select (click, drag and drop) and area that contains the object, then perform a right-click onFill... > Content based > OK .
  • Patch tool: it works very much like the lazo tool, but after selecting the area, you need to drag it on another area in the photo that will be used as base for the fill.

Enhancing details

The best tools to do so are Screen and Burn. All you have to know is that screen lightens up colors and burn darkens them. You just have to use these brushes to draw upon the area of your choice.

If you don't want to directly edit the photo's layer, you can create a new layer by clicking Layer > New > Layer and set the blend mode to Overlay. After that, you need to select the option Fill with blabla... (grey 50%). This adds an invisible gray layer on top of the photo. This layer is where you must use the screen and burn brushes.

Working with clipping masks

Masks are one of Photoshop's main features. A mask controls a layer's opacity to which it is attached. This is done by filling areas on the mask with grayscale colors, remembering that white -> opacity = 100% and black -> opacity = 0%.

In order to add a clipping mask to a layer, you have to select that layer and click on the icon Add clipping mask in the bottom part of the layers panel (it's a white rectangle with a black circle in it). Now you should be able to see the mask preview near the layer preview. In order to know if you're working on the layer or on its mask, you have to pay attention to which preview is highlighted.

Mask are used maily for adjustment layers, as they let us apply them only on some areas of the photo.

Saving your work

When saving your work you should choose an appropriate format.

JPEG and other common extensions for images don't support Photoshop layers. If you want to mantain such informations (for next editings), you should save a .PSD file or a .PDF.

Saving in .jpg means that all the editing will be applied to a source file (more or less like sqeezing all layers into one) and you will not be able to modify a particular layer anymore. This is good if you think this is the definitive version of your image and you're going to print it or to share it on Instagram.

It is a common practice to mantain both a JPG and a PSD.

Based on the final use of the image, it could be useful to resize the file. As an example, you don't need a 5000*7000px 300ppi photo if you just want to share it on Instagram: 1920px on the long side are enough. To resize an image, click Image > Image size and set the desired values.
