Honeypot Lab - Discovering honeypots

Project information

  • Category: Cybersecurity
  • Context: MSc in Computer Science, Network Security course
  • Project date: May, 2020

Installation and conﬁguration of low-interaction honeypots (Cowrie and Dionaea) on Linux virtual machines. Analysis of their features and functionalities through tests conducted with attacking machines on multiple communication protocols. Conﬁguration of a firewall using iptables for the construction of a small ad hoc network to test the proper functioning of the honeypots.
