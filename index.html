UNIVERSITY PROJECTS

  • BSc in Computer Science (2016 - 2019)
    • Course: Operating Systems - Development of a shell for UNIX systems. The shell supports piping and logs the operations (and errors) performed.
      See it on Github!
    • Course: Introduction to Web Development - Development of a webapp to manage shared shopping lists. The system is able to send notifications for items purchased regularly, and it offers geolocalization services in order to notify the user if he/she is near a shop that could sell the listed goods. The back-end was developed in Java (as requested).
    • Course: Human-Computer Interaction - Development of an app prototype for sustainable mobility. The project started with on-field data gathering and a design thinking phase, then went on with the development of a low-fidelity prototype. After an evaluation phase, we were able to develop a medium-fidelity prototype.
      Read more about this!
  • MSc in Computer Science (since 2019)
    • Course: Network Security - In the final group work we were asked to teach a practical lecture about honeypots. In order to analyse and show different kinds of honeypots and their characteristics, we set up a small ad hoc network of virtual machines (managed via iptables). The honeypots used were Cowrie and Dionaea.
      Read more about this project!
    • Course: Multimedia Data Security - Development (in MATLAB) of a robust technique for watermark embedding in grayscale images. The technique is based on multiple applications of the discrete wavelet transform and it aims at preserving the image's quality. On the other hand, also we developed also attacks that aimed at destroying embedded watermarks without causing too much damage to the images.
